# Tab Name Dimmer

A simple client-side Fabric mod for **Minecraft 26.1.2**.

## Feature
When the player list (Tab menu) is open, holding `Shift` will automatically dim every player's nickname on the list, **except** for the specific names you have whitelisted in the configuration. 

This is highly useful for streamers or players on crowded servers who want to quickly highlight specific friends or staff members in the Tab list while fading everyone else into the background.

## Configuration
You can configure the whitelist of names through the **Mod Menu** config screen in-game.

Alternatively, you can manually edit the config file located at:
`config/tab-name-dimmer.json`

## Installation
1. Install [Fabric Loader](https://fabricmc.net/) and Fabric API for Minecraft 26.1.2.
2. (Optional but recommended) Install [Mod Menu](https://modrinth.com/mod/modmenu) to configure the mod in-game.
3. Drop the compiled mod `.jar` into your `mods/` folder.
